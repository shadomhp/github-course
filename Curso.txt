git status - Verifica o status dos arquivos

git add curso.txt

git commit -m "Adicionado Curso.txt"

git log - Log dos commits mostra a Hash de cada commit

git shortlog - Log dos commits por ordem alfabetica e resumido
git shortlog -sn - nr de commits e quem commitou
git log --graph - Mostra em forma grafica os comandos

git diff - o que foi modificado nos arquivos antes de commitar
git diff --name-only --> Somente o nome dos arquivos modificados
git commit -am "Edit" --> Qdo já existe o arquivo


Reset dos erros
git checkout Curso.txt - feito antes do Add

git reset HEAD Curso.txt

git reset --soft  <hash>  --> Pegar as modificações e voltar o commit mas vai para o staged pronto pra commitar
git reset --mixed <hash>   --> Pegar as modificações e voltar o commit mas vai para o modified
git reset --hard  <hash>  --> Pegar as modificações e volta para o unmodified
* Sempre colocar o hash do commit que eu quero voltar.
Somente se não for enviado para o repositorio remoto.

Ligação ao Repositorio remoto
Criação Novo Repositorio
echo "# github-course" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/shadomhp/github-course.git
git push -u origin master

Fazer a ligação do repositorio já criado para o remoto
git remote add origin https://github.com/shadomhp/github-course.git
git push -u origin master

git remote  --> Mostra que já tem uma ligação ao repositorio remoto apelidado Origins
git remote -v   --> Diz mais informações sobre

git push -u origin master   --> envia todas as alterações
